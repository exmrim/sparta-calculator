■개요
 - 프로젝트 이름: 계산기 만들기
 - 프로젝트 기간: 2025.02.26 - 2025.02.27
 - 개발 언어: java 

■설명
 : 숫자와 연산을 입력해서 빠르게 사칙연산 계산하여 그 결과를 화면에 표시해줍니다.

■사용 방식
 1) 프로그램을 실행시킵니다.
 2) 실행시키려는 계산기 프로그램 번호를 입력합니다.
 4) 출력 메세지에 맞게 첫번째 숫자를 입력합니다.
 5) 이어서 두번째 숫자를 입력합니다.
 6) 다음으로 원하는 사칙 연산 기호를 입력합니다.
 7) 입력한 숫자와 기호를 통해 계산식과 결과값이 화면에 표시됩니다.
 8) 이어서 계산을 하시려면 아무 키를 입력합니다.
 9) 종료를 원하시면 'exit' 를 입력합니다.

■폴더 설명
 1) Lv1 : 클래스 없이 기본적인 연산을 수행하는 계산기 폴더
 2) Lv2 : 클래스를 적용해서 기본적인 연산을 수행하는 계산기 폴더
  - App : 계산기 실행시키는 메인 클래스
  - Calculator : 사칙 연산을 수행하는 클래스
 3) Lv3 : 예외처리 및 다른 기능을 추가해서 만든 계산기 폴더
  - App : 계산기 실행시키는 메인 클래스
  - ArithmeticCalculator : 사칙 연산을 수행하는 클래스
  - ExceptionHandler : 예외처리 클래스
